# Animation DSL Compiler

## Overview

This project implements a custom Domain Specific Language (DSL) Compiler for creating web animations.

The compiler converts animation scripts into HTML and CSS automatically.

## Features

- Custom DSL Syntax
- Tokenizer
- Parser
- AST Generation
- Code Generator
- HTML Output
- CSS Animation Generation
- SVG Wave Generation
- Boat Animation Support

## Technologies

- JavaScript
- Node.js
- HTML
- CSS

## Folder Structure

compiler.js
index.html
styles.css
sample.dsl

## How to Run

npm install

node compiler.js

Paste DSL code

Generated files:

- index.html
- styles.css
